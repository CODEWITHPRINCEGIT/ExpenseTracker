#Expense Tracker (C++)

A simple Command Line Expense Tracker built using C++.
This application allows users to add, view, delete, and store expenses using file handling. The program saves expense data so that it persists even after closing the application.

This project is useful for beginners who want to learn practical C++ programming beyond DSA, including file handling, vectors, classes, and modular programming.

-Features
Add new expenses
View all saved expenses
Delete an expense
Automatically save expenses to file
Load saved expenses when the program starts
Simple command line interface

-Technologies Used
C++
STL (Vector)
File Handling (fstream)
Object-Oriented Programming
